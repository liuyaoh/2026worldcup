## 02 使用指南

本系统目前有三条主线能力：

- 赛前预测（Pre-match）
- 比赛中动态预测（Live）
- 历史回放模式（Replay）

### 1）赛前预测

1. 进入预测页（`http://localhost:3000`）
2. 选择主队（Team A）和客队（Team B）
3. 可选填写“特殊因素 / 附加说明”
4. 点击“开始预测”

输出会包含：

- 主胜 / 平局 / 客胜概率
- 战术分析
- 关键胜负手
- 终局判断

### 2）比赛中动态预测（手动赛况输入）

在“比赛中动态预测”区域，输入：

- 当前分钟
- 当前比分
- 红黄牌
- 控球率
- 射正
- xG
- 危险进攻
- 伤病/被动调整
- 场上势头

系统会基于赛前基线实时重算胜率，并解释变化原因。

### 3）事件时间线控制台（半自动）

你可以用事件按钮记录：

- 进球、红黄牌、伤病、射正、高质量机会、危险进攻、场上势头

每次添加/删除事件后：

- 系统会自动更新赛况面板
- 自动触发动态胜率重算

> 这条路径适合“你知道发生了什么事件，但不想手填一堆数字”的场景。

### 4）真实直播模式

真实直播由 `football-data` 提供（见 [03 真实直播接入（football-data）](./03-真实直播接入（football-data）.md)）。

### 5）历史回放模式

当没有真实直播比赛时，用它来验证“事件自动推进 → 胜率自动变化”（见 [04 历史回放模式](./04-历史回放模式.md)）。
