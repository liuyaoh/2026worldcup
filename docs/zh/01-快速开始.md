## 01 快速开始

### 环境要求

- Node.js 18+
- Python 3.11

### 1）配置环境变量

在 `world_cup/` 目录下复制并编辑：

```bash
cp .env.example .env
```

建议至少配置：

```env
# LLM（OpenAI 兼容格式）
LLM_API_KEY=your_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL_NAME=qwen-plus

# Zep（用于图谱）
ZEP_API_KEY=your_zep_api_key

# 真实直播数据源（football-data 免费版建议 30 秒或更高）
LIVE_DATA_PROVIDER=football-data
LIVE_DATA_POLL_INTERVAL_SECONDS=30
FOOTBALL_DATA_API_KEY=your_football_data_api_key_here
FOOTBALL_DATA_BASE_URL=https://api.football-data.org
FOOTBALL_DATA_WORLD_CUP_CODE=WC
```

说明：

- `.env` 里包含密钥，不要提交到仓库，也不要公开。
- 如果暂时没有真实直播比赛，推荐先用“历史回放模式”验证动态胜率链路（见 [04 历史回放模式](./04-历史回放模式.md)）。

### 2）安装依赖

在 `world_cup/` 目录执行：

```bash
npm run setup:all
```

### 3）启动服务

```bash
npm run dev
```

访问地址：

- 前端：`http://localhost:3000`
- 后端：`http://localhost:5001`

### 4）验证是否启动成功

- 打开前端页面能看到“世界杯球队胜率预测系统”
- “历史回放模式”能载入并开始自动回放（最适合用来验证“胜率会自动变化”）
